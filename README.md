 

# 🌙 LumiFix AI – Smart Image Enhancement Tool

> A modern **Digital Image Processing (DIP)** web application that enhances low-light and noisy images using advanced image processing techniques.

---

##   Features

✨ Upload and enhance images in real-time
✨ Improve brightness and contrast
✨ Reduce noise while preserving edges
✨ Apply sharpening filters
✨ Interactive sliders for fine control
✨ Before vs After comparison view
✨ Download enhanced images
✨ Beautiful, modern UI with smooth animations

---

## 🧠 Digital Image Processing Concepts Used

* **Histogram Equalization** → Enhances contrast
* **Gaussian Filtering** → Reduces noise
* **Bilateral Filtering** → Smoothens while preserving edges
* **Image Sharpening (Convolution)** → Enhances details

## ⚙️ Installation & Setup

### 🔹 1. Clone the Repository

```bash id="x6yq1r"
git clone https://github.com/your-username/lumifix-ai.git
cd lumifix-ai
```

---

### 🔹 2. Backend Setup

```bash id="k9z3b1"
cd backend
pip install -r requirements.txt
python app.py
```

---

### 🔹 3. Frontend Setup

```bash id="z8m2v4"
cd frontend
npm install
npm start
```
